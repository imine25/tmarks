# 部署与配置指南

## 浏览器扩展部署

### 开发环境

#### 环境要求
- Node.js 18+
- pnpm 8+

#### 安装依赖

```bash
cd tab
pnpm install
```

#### 开发模式

```bash
# 开发模式（热重载）
pnpm dev

# 指定浏览器
pnpm dev:chrome
pnpm dev:firefox
pnpm dev:edge
```

#### 构建生产版本

```bash
# 构建所有浏览器版本
pnpm build

# 构建特定浏览器
pnpm build:chrome
pnpm build:firefox
pnpm build:edge

# 构建并打包
pnpm pack
```

### 加载扩展

#### Chrome/Edge
1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `tab/dist/chrome` 目录

#### Firefox
1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `tab/dist/firefox/manifest.json`


### 多浏览器构建

#### 构建脚本

```javascript
// scripts/build-multi-browser.mjs
import { build } from 'vite';
import fs from 'fs-extra';

const browsers = ['chrome', 'firefox', 'edge'];

for (const browser of browsers) {
  console.log(`Building for ${browser}...`);
  
  // 复制 manifest
  const manifest = await fs.readJSON('./manifest.json');
  
  // Firefox 特殊处理
  if (browser === 'firefox') {
    manifest.background = {
      scripts: ['background.js']
    };
    delete manifest.background.service_worker;
  }
  
  // 构建
  await build({
    configFile: './vite.config.ts',
    build: {
      outDir: `dist/${browser}`
    }
  });
  
  // 写入 manifest
  await fs.writeJSON(
    `dist/${browser}/manifest.json`,
    manifest,
    { spaces: 2 }
  );
}
```

### 性能优化

#### 缓存策略

```typescript
// functions/_middleware.ts
export async function onRequest(context) {
  const response = await context.next();
  
  // 静态资源缓存
  if (context.request.url.includes('/assets/')) {
    response.headers.set('Cache-Control', 'public, max-age=31536000, immutable');
  }
  
  // API 响应缓存
  if (context.request.url.includes('/api/')) {
    response.headers.set('Cache-Control', 'private, max-age=0, must-revalidate');
  }
  
  return response;
}
```

## 故障排查

### 常见问题

#### 扩展无法加载
- 检查 manifest.json 格式
- 检查权限配置
- 查看浏览器控制台错误

#### API 请求失败
- 检查 API URL 配置
- 检查 API Key 是否有效
- 检查网络连接
- 查看 Network 面板

#### 数据库连接失败
- 检查 wrangler.toml 配置
- 检查数据库 ID 是否正确
- 运行数据库迁移

#### R2 上传失败
- 检查存储桶名称
- 检查 CORS 配置
- 检查文件大小限制
