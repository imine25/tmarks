# TMarks 项目概览

## 项目简介

TMarks 是一个现代化的智能书签管理系统，结合 AI 技术自动生成标签，让书签管理变得简单高效。

## 核心特性

### 智能书签管理
- AI 自动标签推荐
- 多维度筛选
- 批量操作
- 拖拽排序
- 去重检测
- 快照版本管理

### 标签页组管理
- 一键收纳标签页
- 智能分组
- 快速恢复
- OneTab 风格
- 文件夹分组
- 分享功能

### 公开分享
- 创建个性化书签展示页
- KV 缓存加速
- 密码保护
- 访问统计
- 自定义域名

### 浏览器扩展
- 三种保存模式（书签/NewTab/标签页收纳）
- 快速保存书签
- AI 推荐标签
- AI 推荐文件夹
- 离线支持
- 自动同步
- 新标签页替换
- **Widget 系统（8 种组件）**：
  - 快捷方式 Widget
  - 书签文件夹 Widget
  - 天气 Widget
  - 时钟 Widget
  - 待办事项 Widget
  - 备忘录 Widget
  - 热搜榜 Widget（百度/微博/知乎/Bilibili）
  - 每日诗词 Widget
- 天气组件
- 诗词组件
- 农历日期
- 待办事项
- 备忘录
- 热搜榜（4 种热搜源）
- 批量编辑
- 多主题支持
- 国际化（中英文）

### 安全可靠
- JWT 认证
- Refresh Token
- API Key 管理
- 数据加密
- 权限控制
- 速率限制
- API 日志记录

## 技术栈

### 前端
- React 18/19
- TypeScript
- Vite 6/7
- TailwindCSS 3/4
- TanStack Query
- Zustand（状态管理）
- Dexie（IndexedDB）
- @dnd-kit（拖拽排序）
- Lucide React（图标）
- DOMPurify（XSS 防护）
- lunar-javascript（农历）

### 后端
- Cloudflare Workers
- Cloudflare Pages Functions
- Cloudflare D1 (SQLite)
- Cloudflare KV (缓存)
- Cloudflare R2 (快照存储)

### AI 集成
支持 8+ AI 提供商：
- OpenAI
- Anthropic (Claude)
- DeepSeek
- 智谱 AI
- ModelScope
- SiliconFlow
- iFlow
- 自定义提供商

## 项目结构

```
aitmarks/
├── tab/                    # 浏览器扩展
│   ├── src/
│   │   ├── background/    # 后台服务
│   │   ├── popup/         # 弹窗界面
│   │   ├── newtab/        # 新标签页
│   │   ├── options/       # 设置页
│   │   ├── content/       # 内容脚本
│   │   └── lib/           # 共享库
│   └── public/
│
├── tmarks/                 # Web 应用
│   ├── src/               # 前端源码
│   ├── functions/         # Cloudflare Functions (后端)
│   ├── migrations/        # 数据库迁移
│   └── public/
│
├── tmarks-landing-page/   # 落地页
│
└── .cankao/               # 参考项目
    ├── nightTab/
    ├── Bonjourr/
    ├── dnd-kit/
    └── ...
```

## 部署架构

```
用户
 ↓
浏览器扩展 (tab/) ←→ Web 应用 (tmarks/)
 ↓                      ↓
IndexedDB/Storage      Cloudflare Pages
 ↓                      ↓
                   Cloudflare D1 (数据库)
                   Cloudflare KV (缓存)
                   Cloudflare R2 (快照)
 ↓                      ↓
AI 提供商 (OpenAI/Claude/DeepSeek/...)
```
