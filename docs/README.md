## 🎯 核心概念

### 三种书签管理方式

**1. Popup 快速保存**
- 保存到 TMarks 服务器
- AI 智能标签推荐
- 支持离线队列

**2. NewTab 浏览器书签同步**
- 实时双向同步
- 一键批量导入
- AI 智能整理

**3. Web 应用管理**
- 多维度筛选
- 批量操作
- 导入导出

### 数据流向

```
浏览器扩展 → Background Service → IndexedDB (本地缓存)
     ↓
TMarks API → Cloudflare D1 (数据库)
     ↓
Cloudflare R2 (快照存储)
```

### AI 功能

- **标签推荐**：保存书签时自动推荐标签
- **文件夹推荐**：NewTab 保存时推荐文件夹位置
- **智能整理**：按域名和热度重新组织书签结构
