### 3. 标签页组管理页面（TabGroupsPage）

#### 3.1 整体布局

**桌面端三栏布局**：
- 左侧：标签页组树形导航（ResizablePanel，默认 240px，可调整 200-400px）
- 中间：标签页组列表和内容（flex-1）
- 右侧：待办事项侧边栏（ResizablePanel，默认 320px，可调整 280-500px）

**移动端单栏布局**：
- 顶部：MobileHeader（标题 + 菜单按钮）
- 左侧导航改为 Drawer 抽屉式（从左侧滑出）
- 底部：BottomNav 移动端导航栏

#### 3.2 左侧树形导航（TabGroupTree）

**功能特性**：
- 文件夹和分组的树形结构展示
- 拖拽排序和移动（支持跨文件夹移动）
- 创建新文件夹按钮
- 重命名分组/文件夹
- 选中高亮显示
- 刷新按钮

**交互细节**：
- 点击分组：在中间列显示该分组内容
- 点击文件夹：显示文件夹下所有子分组
- 拖拽移动：支持同级排序和跨文件夹移动
- 移动端：选择后自动关闭抽屉

#### 3.3 中间内容区域

**顶部工具栏**：
- 标题（仅桌面端显示）
- SearchBar 搜索栏：
  - 搜索输入框（支持防抖，300ms 延迟）
  - 排序选择器（创建时间/更新时间）
  - 批量模式切换按钮
- BatchActionBar 批量操作栏（批量模式下显示）：
  - 已选数量显示
  - 全选按钮
  - 取消全选按钮
  - 批量删除
  - 批量置顶
  - 批量标记待办
  - 批量导出
  - 取消批量模式

**固定标签页区域**（PinnedItemsSection）：
- 仅在非搜索状态下显示
- 显示所有分组中的置顶标签页
- 支持取消置顶

**标签页组网格**：
- 单列网格布局（grid-cols-1）
- 按文件夹分组显示（显示文件夹标题和标签页数量）
- 每个分组卡片包含：
  - TabGroupHeader（分组头部）
  - TabItemList（标签页列表）

**拖拽功能**（DndContext）：
- 同组内拖拽排序
- 跨组拖拽移动
- DragOverlay 拖拽预览
- 激活距离：8px（防止误触）

**空状态**：
- 无分组时：EmptyState（isSearching=false）
- 搜索无结果：EmptyState（isSearching=true）

#### 3.4 右侧待办事项栏（TodoSidebar）

**功能特性**：
- 显示所有标记为待办的标签页
- 按分组分类显示
- 支持取消待办标记
- 支持打开标签页
- 仅桌面端显示（移动端有独立页面）

#### 3.5 核心功能详解

**搜索功能**：
- 防抖搜索（300ms 延迟）
- 搜索范围：分组标题、标签页标题、标签页 URL
- 搜索自动清空（可配置，默认 15 秒）
- 高性能搜索（使用 searchInFields 优化函数）

**排序功能**：
- 按创建时间排序（created）
- 按更新时间排序（updated）
- 使用 useMemo 缓存排序结果

**筛选功能**：
- 选中特定分组：只显示该分组
- 选中文件夹：显示文件夹下所有子分组
- 未选中：显示全部（按文件夹分组）

**批量操作**：
- 批量删除：确认对话框 → 删除选中项
- 批量置顶：将选中项标记为置顶
- 批量待办：将选中项标记为待办
- 批量导出：导出为 Markdown 格式

**拖拽操作**：
- 同组内排序：更新 position 字段
- 跨组移动：调用 moveTabGroupItem API
- 实时更新本地状态
- 失败时自动回滚

**移动对话框**（MoveItemDialog）：
- 选择目标分组
- 移动到目标分组末尾
- 更新源组和目标组的 position

**分享功能**（ShareDialog）：
- 生成分享链接
- 设置分享权限
- 复制分享链接

**域名高亮**：
- 点击标签页：高亮相同域名的标签页
- 再次点击：取消高亮

**确认对话框**（ConfirmDialog）：
- 删除分组确认
- 批量删除确认
- 其他危险操作确认

#### 3.6 性能优化

- **useMemo 缓存**：
  - groupFilteredTabGroups（分组筛选结果）
  - filteredTabGroups（搜索筛选结果）
  - sortedGroups（排序结果）
- **防抖搜索**：300ms 延迟，减少渲染次数
- **局部刷新**：refreshTreeOnly 只刷新左侧树形列表，不影响中间和右侧列
- **批量更新**：Promise.all 并行更新多个项目
- **高性能搜索**：searchInFields 函数优化搜索性能

#### 3.7 响应式设计

- **桌面端**（isDesktop）：
  - 三栏布局
  - 可调整宽度的侧边栏
  - 显示标题
- **移动端**（isMobile）：
  - 单栏布局
  - 抽屉式导航
  - 移动端标题栏
  - 底部导航栏
  - 底部留白（pb-20）防止被导航栏遮挡

4. **用户设置按钮**：
   - 有容器（bg-card, shadow-sm）
   - 图标：User
   - 跳转到设置页面
   - 悬停效果：bg-primary/5, shadow-md

### 4. 可调整宽度面板（ResizablePanel）

#### 4.1 功能特性

- 支持左侧或右侧面板
- 鼠标拖拽调整宽度
- 宽度限制（minWidth - maxWidth）
- 本地存储保存宽度
- 重置按钮恢复默认宽度

#### 4.2 Props 配置

- side: 面板位置（'left' | 'right'）
- defaultWidth: 默认宽度（px）
- minWidth: 最小宽度（px）
- maxWidth: 最大宽度（px）
- storageKey: 本地存储键名

#### 4.3 UI 元素

**调整手柄**：
- 宽度：1px
- 位置：面板边缘
- 鼠标样式：cursor-col-resize
- 悬停效果：bg-primary
- 拖拽时：bg-primary
- 抓手图标（GripVertical）：
  - 悬停时显示
  - 居中定位
  - 带边框和阴影

**重置按钮**：
- 位置：底部居中
- 图标：RotateCcw
- 悬停时显示
- 点击恢复默认宽度

#### 4.4 交互逻辑

**拖拽调整**：
1. 鼠标按下：setIsResizing(true)
2. 鼠标移动：计算新宽度，应用限制
3. 鼠标释放：setIsResizing(false)
4. 保存到 localStorage

**宽度计算**：
- 左侧面板：e.clientX - rect.left
- 右侧面板：rect.right - e.clientX
- 应用 Math.max(minWidth, Math.min(maxWidth, newWidth))

### 5. 抽屉组件（Drawer）

#### 5.1 功能特性

- 从左侧或右侧滑出
- 遮罩层背景
- 延迟卸载（显示关闭动画）
- 阻止背景滚动
- ESC 键关闭
- Portal 渲染到 body

#### 5.2 Props 配置

- isOpen: 是否打开（必填）
- onClose: 关闭回调（必填）
- children: 抽屉内容（必填）
- title: 抽屉标题（可选）
- side: 滑出方向（'left' | 'right'，默认 'left'）

#### 5.3 UI 结构

**遮罩层**：
- 固定定位（fixed inset-0）
- 背景：background/80
- 模糊效果：backdrop-blur-sm
- z-index: Z_INDEX.DRAWER_BACKDROP
- 点击关闭

**抽屉内容**：
- 固定定位（fixed top-0）
- 宽度：280px（最大 80vw）

### 2. 移动端适配策略

**布局调整**：
- 三栏布局 → 单栏布局

- 内边距：3（移动端）→ 6（桌面端）

## 七、组件设计模式总结

### 1. 响应式设计模式

**断点使用**：
- 移动端优先（默认样式）
- sm: 640px（小屏幕调整）
- md: 768px（中等屏幕调整）
- lg: 1024px（大屏幕调整）

**间距调整**：
- 移动端：p-3, gap-2, text-sm
- 桌面端：p-6, gap-4, text-base

**布局切换**：
- 移动端：单栏布局，抽屉式侧边栏
- 桌面端：多栏布局，固定侧边栏

### 2. 状态管理模式
